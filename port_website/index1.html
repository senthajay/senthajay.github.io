Choices and Extras Management system for NewHome Developers

The Choices and Extras Management System is a web application designed to streamline the management of choices and extras for new homes. It serves two primary stakeholders: buyers and builders. Buyers can use the application to select various choices and extras for their new homes, while builders can manage properties, products, and customer information effectively.

Technologies Used: Programming Languages: Python (Flask 3.9.13), HTML5, CSS, Bootstrap (version 4 and above) Database: SQLite3, SQLAlchemy (Python ORM) Installed Python Libraries: Flask, Flask-Mail, Flask-SQLAlchemy, Jinja2, Matplotlib, NumPy, Pandas, Pillow, Plotly, PyOTP, PyParsing, Python-DateUtil, Pytz, Requests, SQLAlchemy, Stripe, Werkzeug Key Contributions: Implemented the backend using Flask to handle requests and manage the database with SQLAlchemy. Designed responsive front-end interfaces using HTML5, CSS, and Bootstrap. Integrated data visualization features using Matplotlib and Plotly. Managed user authentication and security features with PyOTP and Flask-SQLAlchemy. Set up payment processing using Stripe API.
